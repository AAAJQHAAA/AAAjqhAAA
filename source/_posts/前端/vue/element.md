# vue脚手架element-ui按需引入
- 官网教程：`https://element.eleme.cn/#/zh-CN/component/quickstart`
- 安装：
  - `npm i element-ui -S`
  - `npm install babel-plugin-component -D`
- 脚手架中`babel.config.js`配置文件
```js
module.exports = {
  presets: [
    '@vue/cli-plugin-babel/preset'
  ],
  plugins: [
    ['component', {
      libraryName: 'element-ui',   //插件的名字
      style: true,
      styleLibraryName: 'theme-chalk'   //css样式存放目录
    }, 'element-ui']
  ]
}
```
