vue2脚手架element-ui按需引入

约 370 字大约 1 分钟

vueelement-ui

2024-02-04

  • 官网教程：https://element.eleme.cn/#/zh-CN/component/quickstart
  • 安装：
    • npm i element-ui -S
    • npm install babel-plugin-component -D
  • 脚手架中babel.config.js配置文件
module.exports = {
  presets: [
    '@vue/cli-plugin-babel/preset'
  ],
  plugins: [
    ['component', {
      libraryName: 'element-ui',   //插件的名字
      style: true,
      styleLibraryName: 'theme-chalk'   //css样式存放目录
    }, 'element-ui']
  ]
}
  • 按需引入element-ui-components.js
